repocribro

Repocribro is an extensible sifting tool for information from GitHub repositories powered by Python (Flask, SQLAlchemy, ...). It allows the communities to gather their repositories at the one place when joining a classic GitHub organization is not suitable.

This project has started as a semester work during advanced Python course MI-PYT @ FIT CTU in Prague.
